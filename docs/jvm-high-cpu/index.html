12.CPU飙高如何排查

步骤​

  1. 使用 top 命令，然后按H键查看那个线程cpu占用高。（top -Hp +PID可以直接查，有的系统不支持）
  2. 在步骤1中查出占用高的线程后，将线程id转成16进制：printf "%x\n" 56893
  3. 使用 jstack -l 进程ID |grep 线程ID 可以查出相关进程。
    可以用 jstack -l 进程ID > thread.log 导出到文件。
    然后查询刚刚的线程id，这样比较明了。
  4. 查看相关线程正在执行的代码片段，分析代码是否有问题。
