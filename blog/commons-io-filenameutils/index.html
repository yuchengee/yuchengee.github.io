commons-io中Filenameutils关于文件名的方法

2023年12月11日
java
/io
/文件
阅读需 1 分钟

commons-io中 Filenameutils 类提供了很多关于文件名的方法，使用这个类可以减少代码量，提高工作效率。

String filePath = "/home/wd/tempData/aaa.txt";

方法作用结果
FilenameUtils.getFullPath(filePath)获取父目录，看最后一个是否是 //home/wd/tempData/
FilenameUtils.getBaseName(filePath)获取不带后缀的文件名aaa
FilenameUtils.getExtension(filePath)获取文件后缀，没有则为空txt
FilenameUtils.getName(filePath)获取文件名aaa.txt

灵活运用这个类，可以少写很多代码。
